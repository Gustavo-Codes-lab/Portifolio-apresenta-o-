Certificações

DevOps: Construindo e Gerindo Containers

Durante o curso, aprendi os conceitos fundamentais de containers, como criar, executar e gerenciar containers no Docker, a importância das imagens Docker e do Docker Hub, além de criar minha própria imagem e realizar o push para o repositório. Também aprofundei meus conhecimentos em persistência de dados em containers, entendendo como garantir segurança, organização e reutilização das aplicações em ambientes containerizados.

DevOps: Trabalhando com Repositórios

Durante as aulas, aprendi sobre os principais comandos de trabalho no GitHub, como aplicar GitFlow e GitHub Actions, além de lidar com commits, conflitos, issues e PRs. Também descobri como organizar fluxos e projetos com GitHub Actions e a importância da segurança com o Dependabot. Essa jornada de aprendizado foi incrível e me ajudou a evoluir muito nos meus estudos!

DevOps: Tráfego Seguro e HTTPS

Ao longo das aulas, consegui entender de forma bem mais clara como a comunicação web funciona na prática. Aprendi a usar os métodos HTTP com mais segurança e precisão, a identificar e gerenciar recursos corretamente e, principalmente, a implementar comunicações seguras com criptografia via HTTPS. Estudar a evolução do protocolo HTTP também foi um grande diferencial, trazendo uma visão mais madura sobre como construímos sistemas modernos, resilientes e protegidos.

DevOps: Linux CLI e Scripts

Na prática, trabalhei com Linux no dia a dia, uso do terminal, criação de scripts, automação de tarefas e monitoramento de processos, fundamentos essenciais para ambientes DevOps e back-end.
